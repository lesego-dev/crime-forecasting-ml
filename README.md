# Crime Forecasting using Machine Learning and Geospatial Analysis

## Overview

This project was developed as part of my Master's in Artificial Intelligence and focuses on predicting crime trends across South African police stations using machine learning and geospatial analysis.

The system demonstrates an end-to-end machine learning workflow, including data ingestion, preprocessing, model training, forecasting, and visualization. The goal is to generate predictive insights that can support data-driven decision-making in crime prevention and resource allocation.

This project reflects real-world machine learning engineering practices and production-style project organization.

---

## Objectives

- Analyze historical crime data across South African police stations
- Build a machine learning model to forecast crime trends
- Generate station-level crime predictions
- Visualize geospatial crime patterns
- Demonstrate a complete ML pipeline from raw data to prediction outputs
